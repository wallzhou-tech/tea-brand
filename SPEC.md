# 茶韵 TEA - 项目规格说明书

## 1. 项目概述

- **项目名称**：茶韵 TEA 官网
- **项目类型**：茶叶文化品牌单页官网
- **核心功能**：展示茶叶文化、推广茶叶产品、传递东方禅意生活方式
- **目标用户**：追求品质生活、热爱茶文化的中高端消费者
- **设计风格**：东方禅意 + 现代简约

## 2. 设计规范

### 2.1 色彩系统

| 用途 | 颜色名称 | 色值 |
|------|----------|------|
| 主色 | 茶绿 | #4A7C59 |
| 主色浅 | 浅茶绿 | #6B9E7A |
| 辅助色 | 米白 | #F5F0E8 |
| 辅助色深 | 暖灰 | #E8E0D5 |
| 强调色 | 古铜金 | #B8956B |
| 文字主色 | 深茶褐 | #2C2C2C |
| 文字次色 | 茶灰 | #666666 |
| 白色 | 纯白 | #FFFFFF |

### 2.2 字体规范

- **中文字体**：Noto Serif SC (Google Fonts) - 用于标题
- **英文字体**：Inter (Google Fonts) - 用于正文
- **标题字号**：2.5rem - 4rem
- **正文字号**：1rem - 1.125rem
- **行高**：1.6 - 1.8

### 2.3 间距系统

- 基础单位：8px
- 组件间距：24px - 48px
- 区块间距：80px - 120px

### 2.4 动画规范

- **淡入效果**：opacity 0→1, 600ms ease-out
- **悬停微交互**：transform scale(1.02), 300ms ease
- **卡片悬停**：阴影增强 + 上移4px

## 3. 页面结构

### 3.1 导航栏 (Header)

- Logo：左侧，品牌名"茶韵 TEA"
- 导航链接：右侧（茶文化、产品、关于我们、联系方式）
- 响应式：移动端汉堡菜单
- 背景：透明 → 滚动后米白背景

### 3.2 Hero 区域

- 标题：大标题"茶韵"
- 副标题："一叶一世界，一茶一人生"
- 背景：CSS 渐变（茶绿 → 浅茶绿 → 米白）
- 高度：100vh
- 动画：文字淡入 + 上浮

### 3.3 茶叶文化介绍区

- 布局：3列卡片网格

### 3.5 茶文化科普区

- 标题：中英双语 "茶文化科普 Tea Knowledge"
- 布局：图文左右交替
- 内容：3-4篇科普文章

### 3.6 页脚 (Footer)

## 4. 功能需求

### 4.1 交互功能

- 导航平滑滚动到对应区域
- 卡片悬停效果
- 页面滚动淡入动画 (Intersection Observer)
- 移动端汉堡菜单

### 4.2 响应式断点

| 设备 | 断点 | 布局 |
|------|------|------|
| 桌面 | ≥1200px | 3列网格 |
| 平板 | 768px-1199px | 2列网格 |
| 手机 | <768px | 1列堆叠 |

## 5. 技术实现

### 5.1 技术栈

- HTML5 语义化标签
- CSS3 (Flexbox + Grid)
- Vanilla JavaScript
- 无框架依赖

### 5.2 外部资源

- Google Fonts: Noto Serif SC, Inter
- 图标：内联 SVG 或 Unicode 符号

### 5.3 性能优化

- CSS 动画使用 transform 和 opacity
- 图片使用背景色占位
- 延迟加载非首屏内容

- 中文为主，英文为辅
- 语言切换：无（纯展示）
